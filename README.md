[STM32F401 Nucleo FreeRTOS Template](https://gittub.net/e29qwg/nucleo-freertos)
==================================

A Basic FreeRTOS Template for the STM32F401 Nucleo Board using ST HAL Drivers

This package is based on
 + FreeRTOS 8.1.2 and
 + STM32CubeF4.
